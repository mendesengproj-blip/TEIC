# docs/prompts — charters de campanha (histórico)

Cada arquivo aqui é o **charter de uma campanha**: o prompt original com
pré-registro de kill criteria e, quando a campanha fechou, o veredito
preenchido. Eles foram movidos da raiz para cá na reorganização de jun/2026;
**nenhum conteúdo foi alterado**. Os dados e códigos de cada campanha estão em
`results/` (nunca movidos).
