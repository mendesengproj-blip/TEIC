# FN3-3 — f_A dos parâmetros da DEV: derivado ou livre?

> A relíquia precisa de f_A ~ 10¹⁷ GeV (FN3-1/2). A DEV **prevê** esse f_A, ou ele é
> um parâmetro livre? Esta é a tarefa que separa o Veredito A do B.

## A pergunta decisiva

Ω_{m_A} h² ≈ 0.12 exige **f_A ≈ (1–4)×10¹⁷ GeV** (escala GUT). Há duas leituras de
f_A na DEV — e elas dão respostas **catastroficamente diferentes**.

## Leitura 1 — Stückelberg/Higgs: f_A = m_A/e  → MORRE (47–50 ordens curto)

Da relação de massa do vetor (Higgs-Stückelberg, m_A = e·v ⇒ f_A ~ v = m_A/e),
com e = √(4π/137) ≈ 0.303:

```
   m_A = 3.70e-25 eV  →  f_A = m_A/e = 1.22e-33 GeV   (precisa ~1e17 → déficit 50 ordens)
   m_A = 1.00e-24 eV  →  f_A = m_A/e = 3.30e-33 GeV   (déficit 49 ordens)
   m_A = 1.00e-23 eV  →  f_A = m_A/e = 3.30e-32 GeV   (déficit 48 ordens)
   m_A = 1.00e-22 eV  →  f_A = m_A/e = 3.30e-31 GeV   (déficit 47 ordens)
```

Se f_A é a escala de quebra de simetria fixada pela própria massa via o acoplamento
eletromagnético, ela está **47–50 ordens de magnitude abaixo** do necessário. Como
Ω ∝ f_A², a relíquia seria **Ω ~ 0.12 × (10⁻³¹/10¹⁷)² ~ 10⁻⁹⁵** — completamente
desprezível. **Por esta leitura, o misalignment do m_A NÃO é a matéria escura.**

> Isto é fisicamente esperado: m_A e f_A são independentes em campos ultraleves. A
> massa vem de quebra **explícita** (potencial tipo instanton, exponencialmente
> suprimido); f_A vem de quebra **espontânea** (a escala UV). Atá-los por f_A=m_A/e
> com e eletromagnético é a leitura errada para um ULDM.

## Leitura 2 — escala da rede/corte UV  → ordem certa, mas NÃO derivada

Na leitura correta para ULDM, f_A é a escala de quebra espontânea = a granularidade
UV do substrato (a escala da rede/causal-set, naturalmente Planckiana). O que a DEV
**fixa** sobre essa escala:

- **CROSS_RELATIONS_II / CR4:** m²_iso·λ_p ≈ 520 é um número puro **escala-invariante**
  (λ_p = K = 1). Ele fixa a **razão** m_A/√K = √520 ≈ 22.8 — um número puro — mas
  **não** a escala absoluta em GeV.
- Converter m_A (e portanto f_A) em **unidades físicas** ainda exige a escala
  absoluta da rede, que é **declarada não-derivada desde D3D/e11** (mesmo status de
  X₀ e m_A em unidades físicas: calculáveis dado o corte, não previstos).
- A escala natural do corte é Planckiana (10¹⁸–10¹⁹ GeV). A banda que dá Ω~0.12
  (10¹⁷ GeV) está **uma ordem abaixo de M_Pl** — plausível e "natural", mas
  **escolhida**, não imposta pelos dados de galáxias.

```
f_A necessário para Ω=0.12:        (1–4)×10¹⁷ GeV
f_A da rede (corte UV natural):    ~10¹⁸–10¹⁹ GeV (M_Pl), NÃO fixado em GeV
f_A de Stückelberg (m_A/e):        ~10⁻³³–10⁻³¹ GeV (47–50 ordens curto)
```

## Veredito de FN3-3

```
f_A determinável sem parâmetro livre?   NÃO.
  - Stückelberg (f_A=m_A/e): determinado, mas 47–50 ordens errado → relíquia ~0.
  - Escala da rede: ordem certa (Planckiana), mas a escala física absoluta é
    declarada não-derivada (D3D). A razão m_A/√K=√520 é pura; a escala em GeV não é.
f_A estimado:  ~10¹⁷ GeV é REQUERIDO; é uma escala alta LIVRE (não vem das galáxias).
```

**Consequência:** a abundância Ω~0.12 é **alcançável** (FN3-1/2) mas **não derivada
sem escolher f_A**. A massa m_A vem do Paper II (galáxias); f_A não. Isto é
exatamente o **Veredito B** — DM consistente, não derivada a partir de um único
conjunto de dados. Não é A (sem derivação livre-de-parâmetro) e não é C (a relíquia
é perfeitamente compatível com 0.12; nada de inconsistência estrutural).
