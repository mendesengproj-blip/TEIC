# FL3-3 — Collision Analysis

Pre-registered classification applied to all 8 seeds using the radiation-proof
topological-matter content Q_top (smoothed |baryon density| integral):

- **creation** — Q_top rises well above the initial 2-soliton value (→ Verdict A)
- **annihilation** — Q_top collapses toward 0 (→ Verdict B)
- **elastic** — Q_top stays near its initial value (→ Verdict C)

## Result

| quantity | value |
|---|---|
| scenario counts | **annihilation 8 / 8**, creation 0, elastic 0 |
| Q_top peak / Q₀ (late window) | **0.023** (collapses to ~2%, never rises) |
| Q_top half-time | t ≈ 0.68 (fast annihilation) |
| E_field drop | **~51%** (gradient energy released as radiation) |
| max \|B_total\| (all seeds, all times) | **0.068** (conserved at 0) |
| KE_collision / 2 M_Sk c² | **0.0184** |
| death criterion triggered | **YES** (no creation **and** B ≈ 0) |

The energy-density peak count inflates to 29–44 post-collision, but this is **magnon
turbulence, not solitons** — the conserved charge B and the smoothed matter Q_top both show
no matter was created. The verdict uses the radiation-proof observables only.

## Verdict: **B — ANNIHILATION**

The Skyrmion + anti-Skyrmion annihilate completely into magnon radiation. No additional
pair is created. The topological barrier to creating new winding is not overcome — and
cannot be at this energy, because the collision carries only 1.8% of the 2 M_Sk c²
rest-mass cost of a new pair (FL3-1, FL3-5). The pre-registered death criterion is met:
**B_total stays 0 and the topological-matter content does not rise.**
